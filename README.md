# smart-aeroponic-greenhouse

Repository for my thesis project: **Smart Aeroponic Greenhouse – IoT Automation and Climate Control**.

## Purpose
This repository is used to store:
- thesis writing material
- implementation code
- experiment notes and results

## Suggested Structure
- `docs/` thesis chapters, references, and planning notes
- `src/` source code for firmware, backend, or dashboards
- `data/` experimental datasets (raw/processed)
- `results/` figures, tables, and analysis outputs

## Getting Started
1. Clone this repository.
2. Add your thesis materials into the folders above.
3. Track your progress with clear commits by chapter/feature/experiment.
